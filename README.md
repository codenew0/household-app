# 💰 家計管理アプリ

Pythonで作られたデスクトップ家計管理アプリです。月ごとの支出・収入を項目別に記録・集計・分析できます。

## メイン表 — 月次カレンダー形式の家計表

<!-- スクリーンショット: メイン表のアップ。合計行・まとめ行が見えている状態 -->
![メイン表](https://github.com/user-attachments/assets/8eb5eaa6-055d-4c85-930f-f933c985a9dc)

月の日付が縦に並び、支出項目が横に並ぶ表形式で家計を管理します。

**デフォルト項目（列）:**

| 列名 | 説明 |
|------|------|
| 交通 | 電車・バス・タクシーなど |
| 外食 | レストラン・カフェなど |
| 食料品 | スーパー・コンビニなど |
| 日用品・雑貨 | 生活消耗品など |
| 通販 | ネット通販など |
| ゲーム関連 | ゲーム・周辺機器など |
| サブスク | 定期購読サービスなど |
| 家賃・宿泊 | 家賃・ホテルなど |
| 光熱・通信 | 電気・ガス・水道・スマホなど |
| 医療・美容 | 病院・薬局・美容院など |
| イベント | 旅行・冠婚葬祭など |

**表の特殊行:**

- **合計行** — 各項目の月間合計を自動集計
- **まとめ行** — 収入・収支差額・総支出を表示

**土日の色分け:** 土曜日・日曜日は背景色で視覚的に区別されます。

**年・月の切り替え:**
- 画面上部の `◀ ▶` ボタンで年を切り替え
- 年の数字をクリックすると任意の年を直接入力できます
- `01`〜`12` のボタンで月を切り替え
- 現在の年月日に最も近い月のボタンがハイライトされます

## 支出詳細ダイアログ — セルごとの明細入力

<!-- スクリーンショット: 詳細入力ダイアログ（複数行入力されている状態） -->
![支出詳細ダイアログ](https://github.com/user-attachments/assets/b6589b82-72e6-4aad-a350-ae2de1afbbd2)

メイン表のセルをダブルクリックすると、その日・その項目の詳細を入力できます。1つのセルに複数の明細（支払先・金額・メモ）を登録でき、セルの表示金額は自動的に合算されます。

**列:**
- **支払先** — 過去の入力履歴からドロップダウンで補完
- **金額（円）**
- **メモ** — 過去のメモ入力履歴からTABキーで補完

**操作:**
- ダブルクリック / `Space` キーでセルを編集
- `Tab` キーで自動補完（支払先・メモ）
- `Enter` で編集確定
- 最終行に入力すると自動で新しい空行が追加
- `Ctrl+Z` で元に戻す（最大50操作）
- 行のコピー・切り取り・貼り付け・削除（右クリックメニューまたはショートカット）

## 検索

<!-- スクリーンショット: 検索ダイアログ（検索結果が表示されている状態） -->
![検索ダイアログ](https://github.com/user-attachments/assets/0c02dce6-29d9-44a7-b99c-826fe82e80c4)

`Ctrl+F` または画面上部の「🔍 検索」ボタンで全期間のデータを横断検索できます。

**検索対象:** 支払先・金額・メモの部分一致（大文字小文字を区別しない）

**検索結果の機能:**
- 年月日・項目・支払先・金額・メモを一覧表示
- 各列のヘッダーをクリックしてソート（昇順/降順の切り替え）
- 検索結果の **合計金額・平均金額** を下部に表示
- 行をダブルクリックするとメイン表の該当行に直接ジャンプ・選択

## 月間データ詳細

<!-- スクリーンショット: 月間データ詳細ダイアログ -->
![月間データ詳細](https://github.com/user-attachments/assets/7675e045-bce7-4651-ad9d-7d9aed9caecf)

画面右上の「📅 XX月」ボタンをクリックすると、その月の全明細を一覧で確認できます。

**機能:**
- 月間の全取引を一覧表示（年月日・項目・支払先・金額・メモ）
- 各列のヘッダーをクリックしてソート
- 月間の **合計金額・平均金額** を下部に表示
- 行をダブルクリックするとメイン表の該当行に直接ジャンプ・選択
- 同一内容の行はピンク色でハイライト（重複の目印）

## グラフ — 項目別月間推移

<!-- スクリーンショット: グラフダイアログ（折れ線グラフが表示されている状態） -->
![グラフ](https://github.com/user-attachments/assets/b82a76f7-d509-425b-87d1-5b66fbe429c8)

「📊 図表」ボタンをクリックすると、月ごとの支出推移を折れ線グラフで確認できます。

**グラフの種類:**
- **総支出** — 全項目の月間合計推移
- **総収入** — 月間収入の推移
- **各項目** — 項目ごとの月間推移

**操作:**
- 上部のタブで項目を切り替え
- 右上のドロップダウンで対象年を切り替え
- データポイントに金額ラベルを表示

## カスタム列

<!-- スクリーンショット: +ボタンで列追加のポップアップが出ている状態 -->
![カスタム列](https://github.com/user-attachments/assets/81393b7b-081d-4d6c-ac88-7e32851978ac)

表の右端にある **`+`** ボタンをクリックすると、独自の支出項目を追加できます。

- 列名はヘッダーをダブルクリックで後から編集可能
- 列ヘッダーを右クリックで列の削除も可能
- カスタム列のデータは設定ファイルに保存され、再起動後も維持されます

## ツールチップ

<!-- スクリーンショット: セルにマウスオーバーしてツールチップが表示されている状態 -->
![ツールチップ](https://github.com/user-attachments/assets/c147dbd2-0f75-402e-a457-56c9f6ba030c)

メイン表のセルにマウスを乗せると、詳細情報がポップアップで表示されます。

| ホバー対象 | 表示内容 |
|-----------|---------|
| 通常のセル | 支払先・金額・メモの一覧、複数明細は合計も表示 |
| 合計行のセル | その項目の日別内訳と月間合計 |
| まとめ行「収入」| 収入の内訳一覧 |
| まとめ行「支出」| 全項目の支出内訳一覧 |

## キーボードショートカット

### メインウィンドウ

| キー | 動作 |
|------|------|
| `Ctrl+F` | 検索を開く |
| `Ctrl+C` | セルをコピー |
| `Ctrl+X` | セルを切り取り |
| `Ctrl+V` | 貼り付け |
| `Delete` | セルを削除 |
| `Ctrl+Z` | 元に戻す |
| ダブルクリック | 詳細ダイアログを開く |
| `Shift+クリック` | 範囲選択 |
| `Ctrl+クリック` | 複数セル選択 |
| `Ctrl+スクロール` | 横スクロール |

### 詳細入力ダイアログ

| キー | 動作 |
|------|------|
| `Tab` | 自動補完（支払先・メモ）/ 次の行へ移動 |
| `Enter` | 編集確定 |
| `Space` | セルを編集モードにする |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+C / X / V` | コピー・切り取り・貼り付け |
| `Delete` | 行を削除 |
| `Escape` | ダイアログを閉じる |

## データ保存・バックアップ

データは自動でJSONファイルに保存されます。

```
json/
├── settings.json        # カスタム列・支払先履歴などの設定
└── data/
    └── 2025/
        ├── 2025_01.json  # 2025年1月のデータ
        ├── 2025_02.json
        └── ...
```

- 編集のたびに **即時自動保存**
- アプリ終了時に **バックアップファイル** を自動生成
